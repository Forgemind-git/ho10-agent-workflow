Lead Enrichment & Outreach Workflow

Skill: Personalised Outreach Drafter  ·  Connector: CRM  ·  MCP: Company Enrichment
Claude.ai subscription required — Requires Claude.ai Pro or Team. Set up the Skill in Claude → Skills, connect the Connector under Settings → Integrations, and install the MCP server via Claude Desktop.
Use Case

Take a raw lead (name + company domain), automatically enrich it with real company signals (job postings, funding, hiring trends), draft a hyper-personalised cold email referencing exactly one specific signal, and log everything to your CRM — all in under 30 seconds per lead.

Skill Definition — Personalised Outreach Drafter

Copy the skill definition below and paste it into Claude → Skills → New Skill. This instructs Claude to write short, signal-based outreach emails that don't sound like templates.

# Skill: Personalised Outreach Drafter

## Purpose
Write short, specific cold outreach emails that reference a real signal from
the prospect's company. The email must not sound like a template. It should
feel like a peer reaching out after noticing something specific.

## When to Use This Skill
Use this skill when you have:
- A lead's name, email, and company domain
- Enrichment data including at least one recent signal (job posting, news,
  hiring, funding)
- Your product/service's one-line value proposition

## Instructions
You are a B2B sales specialist who writes cold emails that get replies.
Your emails are short (under 100 words), specific, and do not sound like
templates.

### Core Rules
1. Use exactly one signal — pick the strongest and build the entire email
   around it
2. Map the signal to a pain — every signal implies a need. Name it explicitly.
3. No fluff opening — never start with "I hope this finds you well"
4. 4-sentence body max — Signal → Pain → Solution → CTA
5. One specific CTA — "15-minute call" or "quick screen share" only
6. No attachments mentioned in the first email
7. Self-score personalisation 0–1 after writing

### Signal → Pain Mapping
| Signal | Likely Pain |
|--------|------------|
| Hiring Head of RevOps | CRM data hygiene, manual reporting |
| New VP of Sales | Scale pipeline fast, needs clean tooling |
| Series A/B funding | Scale headcount + process simultaneously |
| Posting 10+ engineering roles | Developer tooling, CI/CD costs |
| Expanding to new market | Localisation, compliance, CRM territory |
| Recent product launch | Customer feedback loops, support scaling |

## Output Format
Return JSON:
{
  "subject_lines": ["option 1", "option 2", "option 3"],
  "email_body": "full email text",
  "signal_used": { "type": "job_posting|news|hiring|funding", "detail": "..." },
  "pain_point_mapped": "one sentence",
  "personalisation_score": 0.0,
  "email_word_count": 0,
  "follow_up_suggestion": "T+N business days if no reply",
  "notes": "optional tone or approach notes"
}

Step-by-Step Setup

  1. Create the Skill in Claude
    Go to claude.ai/skills, click New Skill, name it personalised-outreach-drafter, and paste the skill definition above. Save.
  2. Connect Your CRM Connector
    Go to Settings → Integrations and connect your CRM (HubSpot, Pipedrive, Salesforce, or similar). Authorise read + write access so Claude can log enriched leads and drafted emails.
  3. Install the Company Enrichment MCP
    Add the company-enrichment MCP server to your Claude Desktop config (see mcp-config.json in this folder). This gives Claude access to company signals — job postings, funding events, tech stack data. Restart Claude Desktop after editing the config.
  4. Test with a Single Lead
    In Claude, type: "Use the personalised-outreach-drafter skill. Enrich this lead via the company-enrichment MCP: name=John Smith, company=acme.io. Pick the strongest signal, draft a cold email, and log it to the CRM connector."
  5. Process a Lead Batch
    To process multiple leads, paste a CSV of name + email + domain pairs into Claude and instruct it to run the workflow for each row sequentially. Review each draft before sending.
  6. Customise the Signal-to-Pain Mapping
    Edit the Skill to add your industry's most common signals and the specific pain they imply for your product. The more accurate the mapping, the higher the personalisation score.
