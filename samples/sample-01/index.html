Automate the full content creation cycle: discover what's trending in your industry, draft an on-brand LinkedIn or Twitter/X post in your exact voice, then publish it — all without manual research or copy-editing.

Skill Definition — Brand Voice Writer

Copy the skill definition below and paste it into Claude → Skills → New Skill. This is the instruction set Claude uses to write posts in your brand voice.

# Skill: Brand Voice Writer

## Purpose
Draft social media posts (LinkedIn and Twitter/X) from a trending topic, written
in a consistent brand voice: confident, practical, never hype-driven. The post
should hook the reader in the first line, deliver insight, and end with a clear
call to action.

## When to Use This Skill
Use this skill when you have:
- A topic headline + 2–3 sentence summary (from the Trend Research MCP)
- A target platform (linkedin or twitter)
- Optional: your brand's existing post examples for style matching

## Instructions
You are a social media copywriter for a B2B AI company. Take a trending topic
and write a post that sounds like a founder sharing genuine insight.

### Voice Rules
- Tone: Confident, direct, slightly contrarian
- Practical over theoretical
- No buzzwords: avoid "game-changer", "revolutionary", "leverage", "synergy"
- First person plural or second person

### Structure (LinkedIn — 400–600 chars ideal)
[Hook line — bold claim or counterintuitive observation]

[Bridge — connecting hook to topic]

[3–5 bullet insight lines starting with →]

[CTA — 1 question or call to action]

[3–5 hashtags]

### Structure (Twitter — 200–280 chars)
[Hook — bold claim in 1 line]

[1–2 lines of insight]

[CTA or question]

[2–3 hashtags]

### Output Rules
- Never use exclamation marks
- Never start with "I"
- Hashtags go at the end, not inline
- Emoji: max 2, only if they add meaning

## Output Format
Return JSON:
{
  "variant_a": { "text": "...", "character_count": 0, "platform": "linkedin" },
  "variant_b": { "text": "...", "character_count": 0, "platform": "twitter" },
  "topic_used": "exact headline from input",
  "brand_voice_notes": "why this hook was chosen",
  "generated_at": "ISO8601 timestamp"
}

Step-by-Step Setup

  1. Open Claude Skills
    Go to claude.ai/skills, click New Skill, and give it the name brand-voice-writer.
  2. Paste the Skill Definition
    Copy the full skill text above and paste it into the skill body. Save the skill.
  3. Connect the Social Posting Connector
    Go to Settings → Integrations, find or add your social posting integration (e.g. LinkedIn, Twitter/X). Authorise with OAuth so Claude can publish posts on your behalf.
  4. Install the Trend Research MCP
    Add the trend-research MCP server to your Claude Desktop config file. Open claude_desktop_config.json and add the entry shown in mcp-config.json in this sample folder. Restart Claude Desktop to load the MCP.
  5. Run the Workflow in Claude
    In a new Claude conversation, type: "Use the brand-voice-writer skill. Get trending AI topics via the trend-research MCP, pick the top one, draft a LinkedIn post, then post it via the social connector." Claude will chain the three steps automatically.
  6. Customise for Your Brand
    Edit the Skill definition to include your company name, audience, and 2–3 example posts. The more specific, the more accurately Claude will match your voice.
